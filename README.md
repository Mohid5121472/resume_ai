# 🤖 AI Resume & Cover Letter Generator

A professional Django-based web application that generates ATS-friendly resumes and cover letters using intelligent templates. Features a modern dark SaaS-style UI with copy-to-clipboard functionality.

## ✨ Features

- 📝 **Smart Resume Generation** - Creates professional, ATS-friendly resumes
- ✉️ **Cover Letter Builder** - Generates customized cover letters
- 🎨 **Modern UI** - Dark SaaS-style design with smooth animations
- 📋 **One-Click Copy** - Easy clipboard copying for both documents
- 📱 **Responsive Design** - Works perfectly on desktop and mobile
- ⚡ **Fast Processing** - Instant generation with no API delays

## 🚀 Tech Stack

- **Backend**: Django 4.x (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite (default, can be switched to PostgreSQL/MySQL)
- **Deployment**: Ready for Heroku, PythonAnywhere, or any Django hosting

## 📋 Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
- virtualenv (recommended)
